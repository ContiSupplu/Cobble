Changelog

What's new in Loom

Every update, every improvement, every fix — documented here.

v1.5 Obsidian
May 29, 2026
New Features
  • Bedrock Edition support — detect, launch, and manage Minecraft Bedrock from Loom
  • Built-in add-on browser with MCPEDL, CurseForge, ModBay, and Planet Minecraft
  • Smart downloads — .mcaddon, .mcpack, and .mcworld files auto-download and auto-install
  • Fullscreen browsing mode for the embedded add-on browser
  • Dedicated Bedrock sidebar tab with world, resource pack, and behavior pack management
  • Browse worlds with icons, sizes, and last-played dates
  • Quick-access folder buttons for worlds, resources, and behaviors
Improvements
  • Webview popup windows are now intercepted and handled in-place
  • Download interception prevents save dialogs for Bedrock add-ons
  • SVG icons replace emojis throughout the Bedrock interface
v1.4 Diamond
May 29, 2026
New Features
  • Loomie crash detection — detects game crashes and mod errors in real time
  • One-click "Fix with Loomie" — automatically removes conflicting mods
  • Smart blacklist prevents removed mods from being reinstalled
  • Loader-aware performance mods — Fabric mods no longer installed on NeoForge
  • Dependency-aware installation — skips Iris when Sodium is unavailable
  • Loomie auto-diagnoses all errors, not just the first one
v1.3 Redstone
May 28, 2026
New Features
  • Dynamic Island — a floating HUD overlay with health, coordinates, and Spotify
  • Loomie AI companion powered by Gemini — voice and text
  • Spotify integration with playback controls in the Dynamic Island
  • Discord Rich Presence showing game status
  • Player lookup with Crafatar skin previews
  • Changing Room — skin management and preview
v1.2 Gold
May 27, 2026
New Features
  • Mod browser with Modrinth integration
  • One-click mod, shader, and resource pack installation
  • Automatic dependency resolution
  • Instance file explorer with drag-and-drop support
  • Incognito mode for private sessions
v1.1 Iron
May 27, 2026
New Features
  • Instance management — create, clone, and delete instances
  • Fabric, NeoForge, and Quilt mod loader support
  • Auto-update system via GitHub Releases
  • Multi-account support with Microsoft authentication
  • Game log viewer with real-time output
v1.0 Cobblestone
May 26, 2026
Initial Release
  • Loom launcher — the all-in-one Minecraft launcher
  • Java Edition game launching with version management
  • Glassmorphism UI with dark theme
  • Setup wizard for first-time configuration
  • Automatic Java runtime detection and download
  • Microsoft account authentication
